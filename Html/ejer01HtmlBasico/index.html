    Cuerpo principal del contenido

    La etiqueta main en HTML5 una etiqueta semántica. Esto quiere decir que tiene que ver con el significado de lo contenido dentro de ella. Define el contenido principal de una página. Unicamente puede haber una sola etiqueta como esta en todo el documento. El contenido escrito aqui adentro no repite en el resto del sitio (todo lo contrario a los encabezados de pagina, los menues de navegacion y los pie de pagina). Es usado por buscadores para indexar.

    Sección etiquetas semánticas estructurales mas importantes

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADDRESS

    • Las etiquetas semánticas estructurales estan relacionadas directamente con el significado del contenido y no con los estilos ni las posisciones dentro del documento.

    Reglas para etiquetas semánticas:

  • Los distintos tipos de contenidos se pueden asociar a las etiquetas SECTION.
  • Las unidades de contenidos se asocian a las etiquetas ARTICLE. O sea que un SECTION anida multiples ARTICLES.
  • Las etiquetas HEADER y FOOTER pueden ser anidadas dentro de otras etiquetas HEADER o FOOTER. O sea un FOOTER podria anidar otro HEADER y FOOTER.
  • La etiqueta NAV debe ser unica en el documento y corresponde a los elementos de navegacion del documento.
  • Reservar etiqueta h1 para usar dentro de etiquetas top level como main, header o footer.
